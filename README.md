# QuickHand

QuickHand, dışa aktarılmış bir Wi-Fi profil XML dosyasını okuyup profil adı ve parola bilgisini çıkaran küçük bir masaüstü yardımcı aracıdır.

## Ne Yapar

- kullanıcıdan WLAN profile XML dosyası seçmesini ister
- XML içinden ağ adı ve parola alanlarını okur
- sonucu arayüzde gösterir

## Teknolojiler

- C#
- Windows Forms
- XML

## Notlar

- Girdi olarak Windows tarafından oluşturulmuş profil XML dosyası bekler
